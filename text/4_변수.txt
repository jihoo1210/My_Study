1. 변수란 무엇이고 왜 필요한가
- 기억하고자 하는 값을 메모리에 저장하고 재사용하기 위해서 변수를 사용한다.
- 변수: 하나의 값을 저장하기 위해 확보한 메모리 공간 자체 또는 해당 메모리 공간을 식별하기 위해 붙인 이름.

2. 식별자
- 식별자: 어떤 값을 구별해서 식별할 수 있는 고유한 이름
-- 식별자는 값이 아니라 메모리 주소를 기억한다.

3. 변수 선언
- 변수 선언: 값을 저장하기 위한 메모리 공간을 확보하고 변수 이름과 확보된 메모리 주소를 연결해서 값을 저장할 수 있게 준비하는 것.

4. 변수 선언의 실행 시점과 변수 호이스팅
- 모든 선언문은 코드 실행 이전에 소스평가 과정에서 실행된다.
- 모든 식별자는 호이스팅된다.

5. 값의 할당
- 변수 선언은 런타임 이전에 실행되고, 값의 할당은 런타임에 실행된다.
- 변수 선언과 초기화 사이에는 식별자는 undefined의 메모리 주소를 기억한다.

6. 값의 재할당
- 새로운 메모리 공간을 확보하고 그 메모리 공간에 값을 할당한다.

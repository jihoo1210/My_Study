1. 내부 슬롯과 내부 메서드
- 자바스크립트 엔진 내부 로직

2. 프로퍼티 어트리뷰트와 프로퍼티 디스크립터 객체
- 프로퍼티 상태
-- 값 [[value]]
-- 갱신 가능 여부 [[writable]]
-- 열거 가능 여부 [[enumerable]]
-- 제정의 가능 여부 [[configurable]]
-- 접근자 함수를 통해 프로퍼티의 값을 읽을 때 호출 [[get]]
-- 접근자 함수를 통해 프로퍼티의 값을 저장할 때 호출 [[set]] 
--> Object.getOwnPropertyDescriptor(객체 참조, 프로퍼티 키) 메서드를 통해 간접적으로 프로퍼티의 상태를 받을 수 있다.

3. 데이터 프로퍼티와 접근자 프로퍼티
- 데이터 프로퍼티: 키와 값으로 구성된 일반적 프로퍼티.
- 접근자 프로퍼티: 접근자 함수로 구성된 프로퍼티.

3-1. 데이터 프로퍼티
-[[value]], [[writable]], [[enumerable]], [[configurable]]
-- [[value]]는 프로퍼티 값으로 초기화된다.
-- 나머지의 값은 true로 초기화된다.

3-2. 접근자 프로퍼티
- 자체적으로 값을 가지지 않고 다른 데이터 프로퍼티의 값을 읽거나 저장함.
- [[get]], [[set]], [[enumerable]], [[configurable]].

4. 프로퍼티 정의
- Object.defineProperty(객체 참조, 프로퍼티 키, 프로퍼티 디스크립터 객체);
- Object.defineProperties(객체 참조, 프로퍼티 키, 프로퍼티 디스크립터 객체);

5. 객체 변경 방지
> 얕은 변경 방지
- 객체 확장 금지 | Object.preventExtensions()
- 객체 밀봉 | Object.seal()
- 객체 동결 | Object.freeze()

5-1. 객체 확장 금지
- 확장 가능 여부 확인 | Object.isExtensible()
== 프로퍼티 추가 금지

5-2 객체 밀봉
- 밀봉 여부 확인 | Object.isSealed()
== 프로퍼티 추가 및 삭제와 프로퍼티 어트리뷰트 재정의 금지

5-3. 객체 동결
- Object.isFrozen()
== 프로퍼티 추가 및 삭제와 프로퍼티 어트리뷰트 재정의 금지, 프로퍼티 값 갱신 금지
